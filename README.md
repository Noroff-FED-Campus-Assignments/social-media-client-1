## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/Noroff-FED-Campus-Assignments/social-media-client-1.git
```

2. Install the dependencies:

```
npm install
```

3. Build project
```
npm run build
```

4. Run Jest
```
npm run test
```

5. Run cypress
```
npm run test-e2e
```
